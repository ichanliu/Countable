# 2026-05-13

## 建立倒計時 App (React Native + Expo)

- 根據完整設計規格建立了全新的 Countdowns App (Expo SDK 54)
- 專案結構：
  - `app/_layout.tsx` — 根佈局，含 SafeAreaProvider、GestureHandler、EventsProvider、字體載入、SplashScreen 管理
  - `app/index.tsx` — HomeScreen：自訂標頭、FlatList 卡片列表、FAB、Arrange 模式、空狀態
  - `app/add-event.tsx` — 新增/編輯頁面（modal）：標題輸入、CalendarPicker、背景圖片選擇（expo-image-picker + FileSystem 持久化）、刪除二次確認機制
  - `app/widget-guide.tsx` — Widget Guide 頁面（modal）：Widget 預覽、設定步驟
  - `components/EventCard.tsx` — 210px 事件卡片：ImageBackground/LinearGradient 背景、暗色覆蓋漸層、玻璃擬態倒計時徽章、Animated 按壓縮放動畫、Arrange 模式排列按鈕
  - `components/CalendarPicker.tsx` — 完全自訂內聯日曆：月導航、日期選擇、Today 高亮、Jump to Today
  - `context/EventsContext.tsx` — 狀態管理：AsyncStorage 持久化、addEvent/updateEvent/deleteEvent/togglePin/reorderEvents、Pin 邏輯（僅一個可固定）
  - `constants/theme.ts` — 設計系統色碼、漸層、Radius
  - `constants/types.ts` — CountdownEvent 介面、天數計算邏輯、ID 生成

- 已安裝所有必要依賴：expo-router、@expo-google-fonts/inter、react-native-reanimated、expo-linear-gradient、expo-blur、expo-image-picker、expo-haptics、@react-native-async-storage/async-storage、expo-file-system 等
- 已通過 `npx expo export --platform web` 驗證編譯成功（1245 modules, no errors）
- APK 需用戶本地執行 `cd CountdownsApp && npx expo run:android` 或 EAS Build 建置
