## APK 問題排查與修復

**核心問題確認：Debug APK 不包含 JS Bundle**
- `./gradlew assembleDebug` 直接編譯時，React Native JavaScript 程式碼不會自動打包進 APK
- 導致應用啟動後黑屏（沒有任何 JS 可執行）
- Hermes 需要的是 .hbc（位元組碼），不是原始的 .bundle 檔

**解決方案：**
1. 建立 Release 架構 JS Bundle：`./gradlew app:createBundleReleaseJsAndAssets`（成功，1485 modules）
2. 手動複製到 assets 目錄後重建 Debug APK（36MB，但仍無法正常工作）
3. 最終使用 EAS 雲端編譯的 Release APK（82MB，通過 Expo 完整 CI/CD 流程）

**EAS 雲端建置資訊：**
- 帳號：ichanliu
- 專案：@ichanliu/countdowns
- 狀態：finished（5/15 上午 5:01）
